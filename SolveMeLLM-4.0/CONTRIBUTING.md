# Contributing to 4D-Transformer

感谢您对4D-Transformer项目的关注！我们欢迎所有形式的贡献。

## 🤝 如何贡献

### 报告问题

如果您发现了bug或有功能建议，请：
1. 检查是否已有相关Issue
2. 如果没有，创建新的Issue
3. 提供详细的问题描述和复现步骤

### 提交代码

1. Fork本项目
2. 创建功能分支（`git checkout -b feature/AmazingFeature`）
3. 提交更改（`git commit -m 'Add some AmazingFeature'`）
4. 推送到分支（`git push origin feature/AmazingFeature`）
5. 创建Pull Request

### 代码规范

- 遵循PEP 8代码风格
- 添加必要的注释
- 确保代码可以运行
- 添加测试（如果可能）

### 文档贡献

- 完善使用文档
- 添加使用示例
- 改进文档的可读性

## 📝 贡献指南

### 开发环境设置

```bash
# 克隆仓库
git clone https://github.com/yourusername/SolveMeLLM-4.0.git
cd SolveMeLLM-4.0

# 安装依赖
pip install torch transformers datasets tqdm numpy
```

### 测试

在提交代码前，请确保：
- 代码可以正常运行
- 没有语法错误
- 核心功能正常

### 提交信息

请使用清晰的提交信息：
- `feat:` 新功能
- `fix:` 修复bug
- `docs:` 文档更新
- `refactor:` 代码重构
- `test:` 测试相关

## 🙏 致谢

感谢所有为这个项目做出贡献的人！
